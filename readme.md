# ycloud项目架手架模板

使用项目模板初始化出一套快速开发项目的模板

## 如何使用

1. 安装vue-cli
see [如何安装vue-cli](https://github.com/vuejs/vue-cli)
2. 使用vue-cli初始化项目
vue init yonyouyc/ycloud-template <project-name>

project-name 请输入自己的project-name
